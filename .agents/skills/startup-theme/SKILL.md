# UI Standards
- Always use SCSS for styling.
- Use Angular Signals for state management instead of RxJS where possible.
- Ensure every component is "Standalone".
- Branding: Use hex code #your-startup-color for primary buttons.
- Use variables from global.scss file for all common color, spacing, radius, etc...
- Make component for re-useability.
